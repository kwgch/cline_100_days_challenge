# Catch Game

A simple catch game built with HTML, CSS, and JavaScript.

## How to Play

Use your mouse or touch to move the catcher and catch the falling balls.

## Features

*   Score tracking
*   Lives
*   Difficulty scaling
*   Special items

## Technologies Used

*   HTML5
*   CSS3
*   Vanilla JavaScript

## Mobile Support

The game is designed to be playable on mobile devices.
