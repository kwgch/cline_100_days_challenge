# Babylon.js Endless Runner

## Overview
This project is a 3D endless runner game built using Babylon.js, HTML, CSS, and JavaScript. The game is designed to be playable on both smartphones and PCs.

## Features
- Ground streaming
- Lane branching (3 lanes)
- Obstacle pooling and reuse
- Player model with lane movement and jump actions
- Start, game, and game over screens

## Technologies Used
- Babylon.js
- HTML
- CSS
- JavaScript

## Setup
1.  Download or clone the repository.
2.  Open `index.html` in your browser.
