# progress.md

What works; what's left to build; current status; known issues.

**What works:**

*   Basic emoji matching and clearing.
*   Score tracking.
*   Game over condition.
*   Mobile and PC compatibility.
*   Displaying the next emoji pair.

**What's left to build:**

*   Sound effects.
*   Improved matching algorithm (e.g., chain reactions).
*   Difficulty levels.
*   More sophisticated UI.

**Current status:**

The game is in a playable state, but further improvements are needed to enhance the user experience.

**Known issues:**

*   The game may not be perfectly responsive on all mobile devices.
*   The matching algorithm is basic and doesn't support chain reactions.
