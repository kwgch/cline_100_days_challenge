このアプリケーションは、TensorFlow.jsというライブラリを使い、軽量な自然言語処理 AI モデルである DistilBERT を Web ブラウザ上で直接実行するツールです。

ユーザーがテキストを入力すると、DistilBERT モデルがそのテキストをどのように解釈しているか、特に文章中の単語同士がどの程度関連していると判断しているか（アテンションと呼ばれる仕組み）を、グラフ描画ライブラリ D3.js を使って視覚的に表示します。

これにより、AI モデルの内部的な動作、特に文章理解のプロセスの一部をブラウザ上でインタラクティブに確認することができます。
